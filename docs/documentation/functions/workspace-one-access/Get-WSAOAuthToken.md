# Get-WSAOAuthToken

## Synopsis

Get AOuth Token

## Syntax

```powershell
Get-WSAOAuthToken
```

## Description

The `Get-WSAOAuthToken` cmdlets gets an OAuth token from Workspace ONE Access.

## Examples

### Example 1

```powershell
Get-WSAOAuthToken
```

This example retrieves the am OAuth token from Workspace ONE Access
