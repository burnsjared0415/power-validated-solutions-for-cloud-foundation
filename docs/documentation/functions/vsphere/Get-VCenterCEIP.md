# Get-VCenterCEIP

## Synopsis

Retrieves the the Customer Experience Improvement Program (CEIP) setting for vCenter Server

## Syntax

```powershell
Get-VCenterCEIP
```

## Description

The `Get-VCenterCEIP` cmdlet retrieves the the CEIP setting for vCenter Server.

## Examples

### Example 1

```powershell
Get-VCenterCEIP
```
