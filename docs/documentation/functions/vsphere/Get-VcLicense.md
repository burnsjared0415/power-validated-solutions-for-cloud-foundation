# Get-VcLicense

## Synopsis

Get list of licenses in vCenter Server

## Syntax

```powershell
Get-VcLicense
```

## Description

The `Get-VcLicense` cmdlet gets a list of licenses in vCenter Server.

## Examples

### Example 1

```powershell
Get-VcLicense 
```

This example shows how to get a list of licenses in vCenter Server
