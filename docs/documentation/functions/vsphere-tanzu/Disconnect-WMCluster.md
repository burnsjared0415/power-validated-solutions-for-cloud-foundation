# Disconnect-WMCluster

## Synopsis

Disconnect from o the Supervisor Cluster

## Syntax

```powershell
Disconnect-WMCluster
```

## Description

The `Disconnect-WMCluster` cmdlet disconnects from the Supervisor Cluster.

## Examples

### Example 1

```powershell
Disconnect-WMCluster
```

This example disconnects from the Supervisor Cluster
