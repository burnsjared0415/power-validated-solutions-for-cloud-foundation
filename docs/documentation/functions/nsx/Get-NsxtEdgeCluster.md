# Get-NsxtEdgeCluster

## Synopsis

Gets NSX Edge Cluster Id

## Syntax

```powershell
Get-NsxtEdgeCluster
```

## Description

The `Get-NsxtEdgeCluster` cmdlet gets the NSX Edge Cluster Id.

## Examples

### Example 1

```powershell
Get-NsxtEdgeCluster
```

This example returns the NSX Edge Cluster Id
