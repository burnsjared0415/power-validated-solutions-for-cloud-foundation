# Get-vRLIRole

## Synopsis

Get list of roles

## Syntax

```powershell
Get-vRLIRole
```

## Description

The `Get-vRLIRole` cmdlet gets a list of roles in VMware Aria Operations for Logs.

## Examples

### Example 1

```powershell
Get-vRLIRole
```

This example gets a list of roles in VMware Aria Operations for Logs
