# Get-vRLIContentPack

## Synopsis

Get list of installed content packs

## Syntax

```powershell
Get-vRLIContentPack
```

## Description

The `Get-vRLIContentPack` cmdlet gets a list of all content packs installed on VMware Aria Operations for Logs.

## Examples

### Example 1

```powershell
Get-vRLIContentPack
```

This example gets a list of all content packs
