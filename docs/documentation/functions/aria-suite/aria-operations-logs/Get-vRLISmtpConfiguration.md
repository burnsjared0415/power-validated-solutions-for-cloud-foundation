# Get-vRLISmtpConfiguration

## Synopsis

Get SMTP server settings

## Syntax

```powershell
Get-vRLISmtpConfiguration
```

## Description

The `Get-vRLISmtpConfiguration` cmdlet gets the SMTP server configuration in VMware Aria Operations for Logs.

## Examples

### Example 1

```powershell
Get-vRLISmtpConfiguration
```

This example gets the SMTP settings
