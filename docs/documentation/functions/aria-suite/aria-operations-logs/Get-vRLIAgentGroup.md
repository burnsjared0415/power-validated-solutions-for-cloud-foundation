# Get-vRLIAgentGroup

## Synopsis

Get list of agent groups

## Syntax

```powershell
Get-vRLIAgentGroup
```

## Description

The `Get-vRLIAgentGroup` cmdlet gets a list of agent groups.

## Examples

### Example 1

```powershell
Get-vRLIAgentGroup
```

This example gets a list agent groups
