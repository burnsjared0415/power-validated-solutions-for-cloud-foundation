# Get-vRSLCMHealth

## Synopsis

Check VMware Aria Suite Lifecycle Health Status

## Syntax

```powershell
Get-vRSLCMHealth
```

## Description

The `Get-vRSLCMHealth` cmdlet checks VMware Aria Suite Lifecycle Health Status.

## Examples

### Example 1

```powershell
Get-vRSLCMHealth
```

This example checks VMware Aria Suite Lifecycle Health Status
